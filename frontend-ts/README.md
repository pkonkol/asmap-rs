# frontend-ts

React + Vite frontend that mirrors the Rust/Yew UI and stays binary-compatible with the Rust backend via the `wasm-protocol` bincode bridge.

## Requirements
- Node.js 18+
- npm
- Rust toolchain
- wasm-pack (`brew install wasm-pack` or `cargo install wasm-pack --locked`)

## Install
```sh
npm install
```

## Build WASM protocol bridge
```sh
npm run build:wasm
```

## Run (dev)
```sh
npm run dev
```

## Notes
- The backend must be running at `ws://[::1]:8080/as`.
- If you rebuild `wasm-protocol`, rerun `npm run build:wasm` before `npm run dev`.
- CSS is generated by Tailwind CLI (`npm run dev` also starts the CSS watcher).
